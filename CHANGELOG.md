# История изменений (CHANGELOG)

Все заметные изменения в этом проекте будут фиксироваться в этом файле. Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.1.0] - 2026-05-27

### Добавлено
*   **Новые жесты (Mobile):** Долгое нажатие (400мс) для открытия заметок (peek mode) к задаче на мобильных устройствах, вместо конфликтующих hover-событий.
*   **Редизайн Настроек:** Вкладка управления категориями переведена из формата таблицы в компактный список с отдельным вложенным Bottom Sheet окном для редактирования.

### Изменено
*   **Главный экран:** Радикальный редизайн с фокусом на пузырьковую диаграмму (Bubble Chart). Диаграмма теперь занимает всё свободное пространство.
*   **Добавление задач:** Громоздкая форма заменена на компактную строку с "умным" добавлением. Расширенные настройки перенесены в модальное окно.
*   **Списки и фильтры:** Текстовый список задач скрыт по умолчанию. Фильтры превращены в горизонтальную скроллируемую карусель. Управление зумом перенесено в нижнюю плавающую панель для предотвращения перекрытия графика.
*   **Мобильная адаптивность:** Использование `100dvh` вместо `100vh` для точного соответствия высоте экрана мобильных браузеров. Исправлены проблемы с горизонтальным переполнением на узких экранах.

### Исправлено
*   Баг с "залипающими" тултипами при тапах на мобильных устройствах (переход на Pointer Events + отключение системных callout-меню).
*   `TypeError: toFixed is not a function` при рендеринге значений подкатегорий.

## [1.0.0-mvp] - 2026-05-26

### Добавлено
*   **Движок приоритизации:** Математическая модель расчета веса задач (перенесена из Vue 2 референса и улучшена).
*   **Визуализация:** Интерактивная пузырьковая диаграмма (Bubble Chart) с поддержкой масштабирования.
*   **Google OAuth 2.0:** Система авторизации через социальную сеть для облачного хранения данных.
*   **Изоляция данных:** Полная приватность пользовательских данных (tasks, categories, settings) на уровне бэкенда.
*   **Импорт/Экспорт:** Поддержка импорта JSON-бэкапов из оригинального файла `daily.html`.
*   **CI/CD:** Автоматическое тестирование и деплой на сервер через GitHub Actions и SSH.
*   **Брендинг:** Новый дизайн интерфейса в стиле Apple, включая логотип и favicon.

### Техническое
*   Переход на Laravel 13 и Vue 3 (Composition API).
*   Настройка Docker-окружения для разработки.
*   Реализация 100% покрытия тестами критической бизнес-логики (PHPUnit + Vitest).
*   Внедрение Pinia для управления состоянием фронтенда.
