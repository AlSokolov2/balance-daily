**Баланс.Дейли** — это продвинутый менеджер личных задач и жизненного баланса, перенесенный на современный технологический стек. Система объединяет классический таск-трекинг с мощным математическим движком приоритизации и уникальной визуализацией в виде "пузырьковой диаграммы".

## 🚀 Основные возможности

*   **Умная приоритизация:** Автоматический расчет веса задач на основе категорий, уровней важности, дедлайнов и просроченных дней.
*   **Динамический баланс:** Система автоматически повышает приоритет сфер жизни, которым вы уделяете недостаточно внимания.
*   **Интерактивная визуализация:** Пузырьковая диаграмма на базе **Web Workers** для плавной работы даже с сотнями задач.
*   **Тёмная тема (Dark Mode):** Полная поддержка тёмного оформления с автоматическим переключением.
*   **PWA (Progressive Web App):** Установка приложения на смартфон или десктоп как нативного ПО.
*   **Офлайн-оболочка:** Мгновенная загрузка приложения даже без доступа к сети.
*   **Real-time Sync:** Автоматическое обновление приоритетов каждую минуту и автоматический "Новый день" в полночь.
*   **Безопасность (SSE):** Шифрование всех личных данных в базе данных (AES-256).
*   **Handheld-First дизайн:** Оптимизировано для управления одной рукой, поддержка жестов (pinch-to-zoom, свайпы).

## 🛠 Технологический стек

*   **Backend:** PHP 8.4+, Laravel 13, Sanctum (API Auth), Socialite (OAuth).
*   **Frontend:** Vue 3 (Composition API), Pinia (State), Vite 6, Tailwind CSS 4.
*   **Infrastructure:** Service Workers (PWA), Web Workers (Math), IndexedDB (Planned).
*   **DevOps:** Docker, GitHub Actions (CI/CD), SSH Deployment.

## 🏁 Быстрый старт

### Локальная разработка (Docker)

1. Клонируйте репозиторий.
2. Создайте файл `.env` из примера: `cp .env.example .env`.
3. Поднимите окружение:
   ```bash
   docker-compose up -d
   ```
4. Установите зависимости и запустите сборку:
   ```bash
   npm install && npm run build
   ```
5. Приложение доступно по адресу: [http://localhost:8000](http://localhost:8000)
