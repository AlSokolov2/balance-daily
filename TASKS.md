# 📋 Список задач (Roadmap) — Balance.Daily

Этот файл содержит план развития проекта, разделенный по ключевым направлениям. Задачи могут выполняться в произвольном порядке.

## 📱 1. Мобильный опыт и отзывчивость (Mobile Experience)
- [ ] **PWA (Progressive Web App):** Настройка Service Worker и манифеста для возможности установки на телефон и работы в офлайне.
- [x] **Реакция на ориентацию:** Оптимизация интерфейса и автоматический пересчет диаграммы при повороте экрана (Portrait/Landscape).
- [x] **Вертикальная адаптация:** Улучшение алгоритма упаковки пузырьков специально для узких вертикальных экранов (портретный режим).
- [x] **Жестовое управление:** Добавление новых жестов (свайпы для переключения экранов, "щипок" для масштабирования диаграммы).
- [x] **Оптимизация взаимодействия:** Общая доработка "отзывчивости" элементов на тач-события (устранение задержек, обратная связь).

## 🎨 2. Дизайн и визуальная полировка (Design & UI)
- [x] **Тёмная тема (Dark Mode):** Полноценная поддержка тёмного оформления с автоматическим переключением на основе системных настроек.
- [ ] **Цветовая палитра:** Профессиональный подбор цветов для категорий и интерфейса (создание гармоничной светлой и тёмной темы).
- [x] **Иконографика:** Внедрение системы пиктограмм (иконок) для категорий и действий вместо чистого текста.
- [ ] **Визуальные эффекты:** Реализация плавных анимаций (Transition) при добавлении, выполнении задач и изменении их приоритетов.
- [x] **Оптимизация интерфейса:** Устранение лишнего визуального шума, работа над типографикой и акцентами.

## 🧠 3. Основная логика и продвинутые функции (Core Logic)
- [ ] **Шифрование данных:** Внедрение сквозного или серверного шифрования личных данных (заголовки задач, заметки), чтобы защитить их при утечке БД или локальных данных.
- [x] **Real-time Sync (Автообновление):** Отказ от ручных кнопок "Обновить" и "Новый день". Автоматический перерасчет приоритетов при смене времени/даты.
- [ ] **Улучшение алгоритма:** Внедрение "экспоненциального затухания" приоритета для старых невыполненных задач.
- [ ] **Динамические веса:** Учет времени суток (некоторые категории важнее утром, другие — вечером).
- [ ] **Поиск:** Глобальный поиск по задачам с автоматическим центрированием/подсветкой нужного пузырька на диаграмме.
- [ ] **Статистика и аналитика:** Экран с графиками "Баланса жизни" и трендами продуктивности за неделю/месяц.
- [ ] **Локализация (i18n):** Полноценная поддержка русского и английского языков в интерфейсе.

## 🏗 4. Качество и процессы (Quality)
- [ ] **GitHub Issues:** Перенос этого списка в Issues репозитория для детального отслеживания прогресса.
- [x] **Тестирование:** Поддержание 100% покрытия тестами для новых функций.
