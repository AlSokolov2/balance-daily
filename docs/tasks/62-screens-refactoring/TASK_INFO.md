# TASK_INFO: Рефакторинг навигации (Модалки -> Экраны)

## Основные сведения
- **ID задачи:** #62
- **Тип:** Архитектурный рефакторинг / UX
- **Приоритет:** Высокий
- **Статус:** Анализ завершен, подготовка к имплементации

## Описание
Переход от системы модальных окон, управляемых через `v-if`, к полноценной системе маршрутизации на базе `Vue Router`. Это решит проблемы с навигацией (кнопка "Назад"), улучшит UX на мобильных устройствах и устранит баги, связанные с потерей фокуса при кликах по оверлею.

## Согласованные решения
1. **Навигационная модель:** Гибридная (Tabs + Stacks).
2. **Tab Bar:** Постоянное нижнее меню с иконками для быстрого доступа (один клик) к разделам: Диаграмма, Статистика, Блокнот, Настройки.
3. **Stacks (Overlays):** Полноэкранные накладывающиеся поверхности для редактирования задач и категорий.
4. **Роутинг:** Hash-mode (`/#/path`) для максимальной совместимости PWA.
5. **Анимации:** Использование Vue Transitions для плавного скольжения экранов.
