# Справочник API

Все эндпоинты требуют авторизации через заголовок `Authorization: Bearer <token>`.

## Аутентификация

*   `GET /auth/google` — Редирект на страницу входа Google.
*   `GET /auth/google/callback` — Обработка ответа Google и выдача токена.
*   `GET /api/user` — Получение профиля текущего пользователя.
*   `POST /api/logout` — Отзыв текущего токена.

## Категории (`/api/categories`)

*   `GET /` — Список категорий пользователя.
*   `POST /` — Создание или обновление категории по `slug`.
*   `PUT /{id}` — Обновление параметров категории.
*   `DELETE /{id}` — Удаление категории.

## Задачи (`/api/tasks`)

*   `GET /` — Список задач пользователя.
*   `POST /` — Создание новой задачи.
*   `PUT /{id}` — Редактирование задачи.
*   `DELETE /{id}` — Удаление задачи.

## Настройки и Данные

*   `GET /api/settings` — Получение словаря настроек (например, `notepad_text`).
*   `POST /api/settings` — Пакетное обновление настроек.
*   `GET /api/export` — Экспорт всех данных пользователя в формате JSON.
*   `POST /api/import` — Полная замена данных пользователя данными из JSON.
